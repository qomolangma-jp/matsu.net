# 松.net 管理画面アクセス方法

## 1. テストログインページにアクセス
ブラウザで以下のURLを開く:
```
http://matsu.localhost/test-login
```

## 2. テストユーザーでログイン
以下のいずれかのボタンをクリック:

### マスター管理者
- **メールアドレス**: master@matsu.localhost
- **権限**: 全学年のユーザーを管理可能
- **機能**: すべての名簿閲覧、承認、CSV出力

### 学年管理者（2018年）
- **メールアドレス**: admin2018@matsu.localhost
- **権限**: 2018年卒業生のみ管理可能
- **機能**: 2018年卒の名簿閲覧、承認、CSV出力

### 学年管理者（2019年）
- **メールアドレス**: admin2019@matsu.localhost
- **権限**: 2019年卒業生のみ管理可能
- **機能**: 2019年卒の名簿閲覧、承認、CSV出力

## 3. 管理画面の機能

### 名簿一覧 (`/admin/users`)
- ✅ 統計情報（総登録者数、承認済み、承認待ち、郵送物不達）
- ✅ 絞り込み検索（キーワード、卒業年度、地区会、承認ステータス、郵送物不達）
- ✅ ソート機能（卒業年度など）
- ✅ ページネーション（50件/ページ）
- ✅ 一覧表示（ID、卒業年度＋回期、氏名、メール、地区会、郵送物、承認ステータス）
- ✅ 承認待ちユーザーへの承認/却下ボタン
- ✅ CSVダウンロード（現在の絞り込み条件そのまま適用）

### CSV出力形式
- UTF-8 BOM付き（Excel対応）
- 項目: ID、卒業年度、回期、姓、名、姓（カナ）、名（カナ）、生年月日、メールアドレス、郵便番号、住所、地区会、郵送物不達、権限、承認ステータス、承認日時、承認者、登録日時

### 承認・却下機能
- **承認**: ユーザーのapproval_statusを'approved'に更新、承認日時・承認者を記録
- **却下**: ユーザーのapproval_statusを'rejected'に更新、理由を入力可能
- **権限制御**: 学年管理者は自学年のみ、マスター管理者は全学年を承認可能

サイドバー下部の「ログアウト」ボタンをクリック

## トラブルシューティング

### 「管理者権限が必要です」エラー
- テストログインページからログインし直してください
- 一般ユーザーは管理画面にアクセスできません

### ページが表示されない
```bash
# Laravelログを確認
docker logs matsu-app

# Caddyログを確認
docker logs matsu-caddy
```

### 承認待ちユーザーがいない
```bash
# 新規登録画面からユーザーを登録
# http://matsu.localhost/register
```
